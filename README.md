# Amazon-sales-Analysis
Amazon Sales Data analysis using Pivot Table and charts
Tools used : My SQL , Excel , Power BI , Google Sheets
## Project Description
Analyzed Amazon sales data for different categories
Created pivot table to show total sales per category
Added a column chart to visualize sales distribution across categories
Formatted data and visuals for professional presentation
## Dataset
Contains columns : Order ID , Product Name ,Category , SubCategory , Sales , Quantity , Profit , Region
